# v0.8.9 GREEN Brief

## Decision

Implement GREEN minimum after validated RED (v0.8.8).

## Rationale

v0.8.8 created 20 RED tests that failed by design. v0.8.9 provides the minimal metadata-only adapter to satisfy all 20 contracts.

## Implementation

Metadata/contract only. Two dataclasses (`OneWayPoissonCouplingMetadata`, `PoissonInputSource`), one builder (`build_poisson_input_source`), and two validators.

## Not Included

* No solver runtime.
* No physical interpretation.
* No calibration.
* No device prediction.
* No new dependencies.
* No paid API, SDK, or goal-like execution.
