# v0.8.1 ADR-013 Acceptance Review

## Core Question

Is ADR-013 mature enough to be promoted from Proposed to Accepted?

## Scope of Acceptance

If accepted, ADR-013 authorizes only:

* future v0.8.2 C3 RED;
* RED tests for conservative grid projection;
* no production code;
* no solver runtime;
* no physical phi;
* no Poisson solve;
* no AI-for-Science runtime.
