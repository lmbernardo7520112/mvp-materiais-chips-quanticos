# v0.5.4 Experimental-Anchored Energy Profile Feasibility Brief

## Core Question

Can experimental or literature evidence guide the P1 piecewise D_it(E)
profile without creating false calibration?

## Decision Options

### Option A — Keep P1 Purely Demonstrative

- **Pros:** Lowest risk; simplest implementation; no source dependency.
- **Cons:** Less physically informative; bins may appear arbitrary; harder
  to justify scale choices in publications.

### Option B — Literature-Informed P1 Bins (E1)

- **Pros:** Improves physical grounding using established D_it(E) shapes
  (U-curve from C-V studies); low/moderate risk; compatible with v0.6.0
  RED; bins traceable to peer-reviewed sources.
- **Cons:** Still not calibrated; source transferability limited to similar
  stacks; shape details depend on measurement technique.

### Option C — Experimental-Profile Prior P1 Bins (E2)

- **Pros:** Strongest physical direction short of calibration; can map
  digitized D_it(E) curves into bins; higher fidelity shape.
- **Cons:** Requires full source metadata; risk of false precision;
  source stack may not match target; must be clearly labeled as prior.

### Option D — Same-Stack Calibration (E3/E4)

- **Pros:** Strongest evidence; genuine device-level grounding.
- **Cons:** Unavailable now; requires experimental campaign; requires
  separate ADR beyond ADR-010; risks blurring C1/C2 boundary.

## Recommendation

1. **Adopt Option B** as default for future v0.6.0 P1 implementation.
2. **Allow Option C** only as optional documented profile when source
   metadata meets all E2 criteria (see evidence taxonomy).
3. **Reject Option D** for current roadmap — E3/E4 require a future ADR.
4. **Keep ADR-010 Proposed** until this feasibility review is incorporated
   into the acceptance process.

## Decision

The next ADR-010 acceptance review (v0.5.5 or later) should include an
appendix requiring **evidence-level labels** (E0, E1, or E2) for any P1
profile used in benchmarks or demonstrations. E3/E4 profiles remain
blocked until a dedicated calibration ADR is proposed and accepted.
