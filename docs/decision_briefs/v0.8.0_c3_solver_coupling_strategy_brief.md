# v0.8.0 C3 Solver Coupling Strategy Brief

## Core Question

Should the project connect C2 directly to the Poisson solver now?

## Answer

No.

The correct next step is to define a conservative grid projection contract before any solver runtime.

## Why Not Direct Solver Coupling Now?

* C3 projection contract does not exist.
* Boundary metadata is not formalized.
* Grid projection has not been tested.
* physical phi remains blocked.
* solver coupling would create overclaim risk.
* AI-for-Science track must remain separate.
* C2 is validated only as isolated mapping/demo.
