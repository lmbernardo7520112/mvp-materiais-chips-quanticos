# v0.7.0 C2 Interface Source / Regularization Evidence Note

## Purpose

Summarize how literature informs the C2 decision without claiming
universal t_eff or calibration.

This document does NOT introduce new sources. It references materials
already documented in the repository where possible.

## 1. Interface Sheet Charge / Jump Condition

**Concept:** σ_eff [C/m²] as interface boundary condition.

**Literature basis:**

- Sze & Ng (2006), *Physics of Semiconductor Devices*, 3rd ed.,
  Chapters 4–5: interface charge modeled as sheet charge at
  Si/SiO₂ boundary in MOS electrostatics.
- Standard in semiconductor device physics textbooks.

**Classification:** Concept support.

**Key equation:**

    n̂ · (ε₂∇φ₂ − ε₁∇φ₁) = −σ_eff

**Note:** This is the standard boundary condition for interface
charge in the Poisson equation across a dielectric interface.

## 2. Conservative Volume Regularization

**Concept:** ρ_reg = σ_eff / l_reg within a thin layer.

**Literature basis:**

- Standard numerical technique in finite-element and finite-
  difference device simulation (Selberherr, 1984; TCAD manuals).
- Used when solver frameworks require volumetric source terms.

**Classification:** Concept support (numerical technique).

**Conservation rule:**

    ∫_V ρ_reg dV = ∫_A σ_eff dA

**Note:** l_reg is tied to mesh resolution (~1–2 cells), not to
physical trap depth.

## 3. Gaussian or Distributed Charge Regularization

**Concept:** Replace delta-function sheet charge with a smooth
spatial distribution (e.g., Gaussian of width w).

**Literature basis:**

- Used in some QTCAD implementations for numerical stability.
- Width w is a numerical parameter, not a physical depth.

**Classification:** Concept support (numerical technique).

**Note:** Not currently recommended for the MVP. If needed, it
would be a variant of C2-B with a smooth profile instead of
uniform l_reg slab. Same conservation and reporting requirements
apply.

## 4. Near-Interface / Border / Oxide Trap Depth Priors

**Concept:** Literature reports on spatial distribution of traps
near the Si/SiO₂ interface.

**Key distinctions:**

| Trap family | Typical depth | Typical technique | Source role |
|-------------|:------------:|-------------------|------------|
| Interface states (D_it) | ~monolayer | Charge pumping, conductance | C1 chain input (already done) |
| Border traps | ~1–3 nm | Noise spectroscopy, charge pumping | Depth prior candidate (C2-C) |
| Oxide traps | >3 nm | Stress-induced leakage | Not relevant for C2 |

**Literature basis (from repository documents):**

- v0.5.2 benchmark: D_it values for Si/SiO₂ (interface states).
- v0.5.4 evidence sources: energy profile shapes.

**Important:** The v0.5.2 benchmark documents D_it **values** (scale),
not D_it **spatial depth**. Border trap depth literature exists but
is NOT currently documented in the repository. If future C2-C work
needs border trap depth priors, new literature review will be required.

**Classification:** Scale envelope (existing), depth prior candidate
(future, not yet sourced).

## 5. Why Generic t_eff Is Rejected

**Arguments:**

1. No universal t_eff for Si/SiO₂ interface traps exists.
2. Literature values span orders of magnitude depending on:
   - trap family (interface vs border vs oxide);
   - measurement technique;
   - process conditions (oxidation, anneal, passivation).
3. Selecting a single t_eff creates false impression of calibration.
4. t_eff conflates numerical regularization with physics.
5. The primary C2-A path does not require any thickness parameter.

## 6. Why Calibration Is Blocked

**Arguments:**

1. The MVP has no experimental validation pipeline.
2. No own-device or same-stack measurements are available.
3. Claiming calibration without experimental data violates:
   - ADR-010 evidence policy (E0 rejected);
   - v0.5.6 amendment (S0 test-only);
   - Physics dimensional audit skill.
4. Calibration requires its own future ADR beyond v0.7.x.

## Classification Rules

Every use of literature in C2 must be classified as:

| Classification | Meaning | Example |
|----------------|---------|---------|
| Concept support | Validates the mathematical formulation | Jump condition from Sze & Ng |
| Scale envelope | Provides order-of-magnitude range | D_it ~10¹⁰–10¹¹ eV⁻¹·cm⁻² |
| Depth prior candidate | Spatial distribution hypothesis | Border trap ~1–3 nm |
| Not calibration | Explicit non-claim | All of the above |
