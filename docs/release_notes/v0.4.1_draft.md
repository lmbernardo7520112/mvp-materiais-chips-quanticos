## Explicit Non-Claims

> [!CAUTION]
> v0.4.1 does **not**:
> - Calibrate any parameter against experimental data
> - Change any parameter value in the code
> - Modify any solver, script, or test
> - Introduce new dependencies
> - Claim device-level prediction capability
> - Use parameters from non-Si material systems as Si proxies
