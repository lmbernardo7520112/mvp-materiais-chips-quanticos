# v0.6.0 — C1 Piecewise D_it(E) Energy Profiles

## Summary

Implements C1 energy-distribution bookkeeping through piecewise D_it(E)
integration, as authorized by ADR-010 (Accepted + v0.5.6 amendment).

## Added

| API | Purpose |
|-----|---------|
| `EnergyInterval` | Single energy bin [J] |
| `PiecewiseDitProfile` | Piecewise-constant D_it(E) container |
| `integrate_piecewise_dit` | N_it = Σ D_i × ΔE_i |
| `compute_sigma_eff_from_energy_profile` | σ_eff from N_it |

## Physics

```
N_it = Σ D_i × ΔE_i   [m⁻²]
σ_eff = s_charge × q_e × N_it × f_occ   [C·m⁻²]
```
