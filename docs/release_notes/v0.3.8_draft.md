# v0.3.8 — v0.4 policy gate activation

## Summary
Activates policy `current_stage: v0.4` after ADR-007 acceptance. This release updates the quality gates only and does not implement Poisson or rho_eff. It strictly prepares the repository governance for the upcoming test-driven implementation.

## Changes
- Updated `tools/quality_gates/policy.json` to `current_stage: v0.4`.
- Enforced strict allowlist for `Poisson` and `rho_eff` files.
- Maintained bans on Schrodinger, TCAD, and predictive coherence claims.
- Updated governance and walkthrough documentation to reflect the new state.
- Added tests to `tests/test_quality_gates.py` for the new allowlist features.
