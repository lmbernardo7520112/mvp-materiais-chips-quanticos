---

## Summary

v0.4.5 is a documentation-only release that reviews whether the passive
SI constants scaffolding introduced in v0.4.4 (`units.py`, `scale_modes.py`)
should be integrated into the runtime as metadata declarations.

**No code was changed.** No src/, scripts/, or tests/ files were modified.

---

## Decision

| Option | Description | Status |
|--------|-------------|--------|
| A | Keep scaffolding passive | Acceptable as current state |
| B | Metadata-only runtime declaration | **Recommended** for future v0.4.6 |
| C | Numerical coupling to solver | **Prohibited** — requires new ADR |
