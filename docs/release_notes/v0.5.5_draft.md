# v0.5.5 — ADR-010 Accepted: C1 Energy-Distribution Scope

## Summary

Accepts ADR-010 for future C1 energy-distribution modeling. Promotes
ADR-010 from Proposed to Accepted after review by three research councils
(v0.5.3, v0.5.4, v0.5.5) and a 15-criteria acceptance review (all PASS).

- Option C (E2 experimental-profile prior) accepted only conditionally.
- E3/E4 calibration-grade anchoring blocked.
