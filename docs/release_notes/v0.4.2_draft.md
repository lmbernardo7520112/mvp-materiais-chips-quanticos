## Key Decisions

### ADR-008 Status: **Proposed**

The ADR proposes:
- A dimensional chain from D_it to φ.
- Three options (A, B, C) for unit introduction.
- A preliminary recommendation of Option B (literature-scaled constants only).
- Future implementation requirements.
- Explicit non-goals.

### Council Recommendation

The five-expert research council voted **unanimously for Option B** as the
next implementation step, conditional on ADR-008 acceptance.

### Critical Risks Identified

| Risk | Severity |
|------|----------|
| R1 — False calibration from SI units | Critical |
| R2 — Confusing normalized domain with physical geometry | Critical |
| R7 — Treating literature range as device-specific calibration | Critical |

## What Comes Next

- **If ADR-008 is accepted:** A future implementation PR will introduce
  Option B (ε = ε_r · ε₀) with explicit mode separation.
- **If ADR-008 is rejected:** The project remains fully demonstrative
  until an alternative approach is proposed.
- **Option C** (full charge closure with physical D_it) is roadmap only,
  deferred until Option B is validated and additional prerequisites
  (ΔE convention, t_eff justification, domain geometry) are met.

## Disclaimers

- v0.4.0 remains a **demonstrative, uncalibrated** model.
- No output of this project should be interpreted as a physical device
  prediction, calibrated measurement, or validated simulation result.
- The model scope is strictly **silicon/CMOS spin-qubit process-to-device
  bridge**, without coherence, fidelity, or charge-noise prediction.
