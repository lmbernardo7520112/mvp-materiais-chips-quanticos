# v0.5.6 — ADR-010 Hardening: No Demonstrative Evidence Profiles

## Summary

Hardens ADR-010 before v0.6.0 by rejecting demonstrative E0 profiles as
operational evidence. Introduces S0 TEST_ONLY for mathematical verification
fixtures.

## Decision

- E0 operational profiles: **rejected** (deprecated).
- S0 TEST_ONLY: allowed only for mathematical verification in tests.
- E1 literature-informed: **minimum operational** evidence level.
- E2 experimental-profile prior: remains conditional with metadata.
- E3/E4: remain blocked.

## Not Implemented

- no code changes;
- no tests;
- no scripts;
- no D_it(E) implementation;
- no rho_eff;
- no t_eff;
- no solver coupling;
- no C2/C3;
- no calibration claims.
