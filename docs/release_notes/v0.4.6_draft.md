## Summary

v0.4.6 implements metadata-only runtime declaration for the existing bridge CSV.
This follows the v0.4.5 council decision (5/5 unanimous Option B) and adds safe
metadata fields to `poisson_bridge_metrics.csv` without changing any numerical
values, solver behavior, or physical equations.

## Added

### Serialization Helpers (`src/mvp_quantum_materials/scale_modes.py`)

- `scale_metadata_to_record(metadata)` ÔÇö serialize ScaleMetadata to flat dict.
- `attach_scale_metadata_to_metrics(metrics, metadata)` ÔÇö merge metadata into
  metrics dict without mutation, with physical interpretation guard.
