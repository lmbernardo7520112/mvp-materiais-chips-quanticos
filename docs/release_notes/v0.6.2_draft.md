# v0.6.2 — Curated E1/E2 D_it(E) Profile Library

## Summary

Adds curated, metadata-rich E1/E2 D_it(E) profile library for C1
energy-profile workflows.

## Added

- `dit_profile_library.py` — curated profile module.
- 3 E1 literature-informed Si/SiO₂ profiles:
  - `si_sio2_literature_nominal` — 3-bin U-shape, D_it 1–5×10¹⁰ eV⁻¹·cm⁻²
  - `si_sio2_literature_high_trap` — 3-bin U-shape, D_it 1–5×10¹¹ eV⁻¹·cm⁻²
  - `si_sio2_literature_low_trap` — 3-bin U-shape, D_it 0.5–1×10¹⁰ eV⁻¹·cm⁻²
- E2 experimental-prior factory (`build_e2_experimental_prior_profile`).
- E1 factory (`build_literature_informed_profile`).
- 20 tests in `test_dit_profile_library.py`.

## Physics

Profiles feed the C1 chain:

    D_it(E) → N_it → σ_eff

Unit conversion from eV⁻¹·cm⁻² to J⁻¹·m⁻² is explicit and uses
ELEMENTARY_CHARGE from units.py.

## Still Blocked

- σ_eff → ρ_eff (requires t_eff, ADR-C2)
- ρ_eff → φ (Poisson coupling, v0.8.0)
- φ → V_conf (v0.8.0)
- Quantum confinement solver (v0.9.0)

## Evidence Policy

- S0_TEST_ONLY: test fixtures only.
- E0 operational: rejected.
- E1: minimum operational.
- E2: conditional with complete metadata.
- E3/E4: blocked.
- All profiles: calibration_status = "not_calibrated".

## Not Implemented

- no ρ_eff
- no t_eff
- no solver coupling
- no quantum confinement solver
- no C2/C3
- no calibration claims
- no coherence prediction
