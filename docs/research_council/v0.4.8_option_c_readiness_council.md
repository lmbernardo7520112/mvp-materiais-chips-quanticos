## Expert 2 — Semiconductor Device Physicist

**Option C ready for implementation?** No.

**Blocking gaps:**
- δE_window is undefined. For Si at room temperature, a reasonable
  choice might be ±2kT around midgap (~0.1 eV), but this depends
  on the trap energy distribution, which varies by process.
- Charge sign convention is missing. At the Si/SiO₂ interface:
  - Above midgap: acceptor-like traps (negative when filled).
  - Below midgap: donor-like traps (positive when empty).
  - Net charge depends on Fermi level, which the current model
    does not compute.
- D_it values span 10⁹–10¹³ eV⁻¹·cm⁻² depending on oxidation
  conditions, annealing, and interface quality. A single representative
  value requires justification and tier classification.

**Equations that must be fixed:**
- N_it = D_it × δE_window (with justified δE_window).
- σ_eff = q_e × N_it × sign_factor (with documented convention).

**Parameters needing tier assignment:**
- D_it: currently no value in code. Needs T2 or T3 with references.
- δE_window: needs definition and T2/T3 classification.
- sign_factor: needs convention document.

**Skills governing this analysis:** physics-dimensional-audit.

## Expert 3 — Computational Materials Scientist

**Option C ready for implementation?** No.

**Blocking gaps:**
- t_eff regularization is physically questionable. Smearing a surface
  charge density (σ_eff) into a volume charge density (ρ_eff = σ_eff / t_eff)
  is a modeling choice, not a physical law. The value of t_eff (currently
  1 nm, tier TX) directly controls the magnitude of ρ_eff.
- The relationship between surface traps and volume charge depends on
  whether we model:
  - A sheet charge at the interface (surface mode).
  - A distributed charge in a thin layer (volume smearing mode).
  These are different physics with different Poisson formulations.

**Sanity checks needed:**
- ρ_eff order of magnitude: for D_it = 10¹¹ eV⁻¹·cm⁻², δE_window ≈ 0.1 eV,
  t_eff = 1 nm → ρ_eff ≈ 10⁵ C/m³. Is this physically reasonable?
- φ magnitude: for a 100 nm domain with ε_Si, ρ_eff ≈ 10⁵ C/m³,
  φ ≈ ρL²/(2ε) ≈ 10⁵ × (10⁻⁷)² / (2 × 10⁻¹⁰) ≈ 5 × 10⁻⁵ V.
  This is in the μV range, which is physically plausible for a small
  defect perturbation but needs validation.

**Skills governing this analysis:** physics-dimensional-audit, scope-guardrails.

## Expert 4 — Applied Mathematician / PDE Specialist

**Option C ready for implementation?** No.

**Blocking gaps:**
- The current solver uses ε = 1 (dimensionless). Replacing with
  ε = ε_r × ε₀ ≈ 1.04 × 10⁻¹⁰ F/m changes the coefficient matrix
  dramatically. The iterative solver (Gauss-Seidel) convergence
  properties will change.
- Boundary conditions must be re-examined. Current Dirichlet BCs
  (φ = 0) are appropriate for a normalized domain but may need
  physical values for a real device geometry.
- The geometry mode (normalized_2d) must be distinguished from
  physical_2d. The Laplacian discretization depends on physical
  grid spacing (dx, dy in meters), not normalized coordinates.

**Minimum tests:**
- Solver convergence with physical ε (should still converge but may
  need more iterations or relaxation parameter adjustment).
- Manufactured solution test with physical ε to verify order of accuracy.
- Boundary condition consistency test.

**Skills governing this analysis:** physics-dimensional-audit, tdd-red-green-release.

## Expert 5 — Numerical Methods Specialist

**Option C ready for implementation?** No.

**Blocking gaps:**
- With ε ≈ 10⁻¹⁰ and ρ ≈ 10⁵, the source term ρ/ε ≈ 10¹⁵.
  This creates a stiff system where the solution magnitude is very
  large relative to the grid spacing. The current Gauss-Seidel solver
  with fixed tolerance may need:
  - Relative tolerance instead of absolute.
  - Under-relaxation for stability.
  - Condition number analysis.
- The current convergence test uses `solver_residual` as an absolute
  measure. With physical units, the residual magnitude changes by
  orders of magnitude. The convergence criterion must be scale-aware.

**Sanity checks:**
- Solver iteration count should not exceed O(N²) for a well-conditioned
  system. Monitor for convergence degradation.
- Residual should decrease monotonically. Non-monotonic residual
  indicates instability.

**Skills governing this analysis:** physics-dimensional-audit, ai-rse-gateops.
