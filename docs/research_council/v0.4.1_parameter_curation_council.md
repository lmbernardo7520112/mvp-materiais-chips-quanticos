## Deliberation Record

### Physical Constants (q, k_B, ε₀)

**Verdict:** q and k_B are CONST_EXACT (SI 2019 defining constants).
ε₀ is CONST_DERIVED: it is derived from ε₀ = 1/(μ₀c²), where c is exact
but μ₀ now carries experimental uncertainty via the fine-structure constant α
(relative uncertainty ~1.5×10⁻¹⁰). For MVP purposes this is negligible,
but ε₀ must not be labeled "exact" in the same sense as q or k_B.

### Semiconductor Parameters (ε_r, D_it)

**Verdict:** T2 candidates identified but NOT used in current code.
ε_r(Si) = 11.7 and D_it = 10¹⁰–10¹² eV⁻¹cm⁻² are well-established.
Primary D_it anchors: Fleetwood (2018), Sze & Ng (2007).
Connors2022 is classified as CONTEXT_ONLY for Si/SiGe charge noise
relevance — NOT as a direct D_it numerical anchor unless explicit
D_it extraction is documented in the source.

### Effective Charge Closure (λ_T, λ_C, C_sat, N_ref, t_eff)

**Verdict:** These remain TX or T3. The closure is Boussinesq-inspired
(ADR-007 Option C-B) and is inherently demonstrative. No physical anchor
exists for the coupling weights.

N_ref is split into two entries:
- **N_ref_active** (TX): the code value 1.0, dimensionless, no promotion.
- **N_ref_to_Dit_mapping** (T1_candidate): conceptual link to q·D_it·t_eff,
  NOT implemented, requires unit conversion infrastructure and device t_eff.

### Defect Parameters (D0_def, E_D, G0, R0)

**Verdict:** D0_def is promoted from T0 to T1 as a literature-inspired
scale parameter. However:
- The SI conversion was corrected: 0.002–10 cm²/s = 2×10⁻⁷–10⁻³ m²/s
  (NOT 10⁻¹⁴–10⁻⁸ as previously stated).
- The MVP value (10⁻⁸ m²/s) is BELOW the Bracht1998 D0 range.
- D0 (Arrhenius pre-exponential) must not be confused with D(T)
  (temperature-dependent effective diffusivity).

E_D remains T1. G0 and R0 remain T0.

### Numerical Parameters (dx, dy, tolerance, max_iter)

**Verdict:** NUM tier confirmed. No physical calibration needed.

## Technical Dissent / Corrections Required

The following corrections were identified during scientific hardening
review and are mandatory before the council approval is unconditional:

1. **D0_def conversion error:** The initial draft incorrectly stated
   0.002–10 cm²/s = 10⁻¹⁴–10⁻⁸ m²/s. Correct value: 2×10⁻⁷–10⁻³ m²/s.
   D0 vs D(T) distinction must be maintained throughout.

2. **ε₀ exactness error:** The initial draft described ε₀ as "exact by
   2019 SI redefinition." ε₀ is NOT a defining constant; it is derived
   and carries experimental uncertainty via α. Corrected to CONST_DERIVED.

3. **N_ref active vs mapping conflation:** The initial draft suggested
   "N_ref TX→T1" as a promotion of the active code parameter. Corrected
   to split: N_ref_active (TX, no promotion) and N_ref_to_Dit_mapping
   (T1_candidate, conceptual only, not implemented).

4. **Connors2022 source role:** Reclassified from parameter_anchor for
   D_it to context_anchor for charge noise relevance, unless explicit
   D_it extraction is documented.

**Council approval is conditional on these corrections being applied.**
All corrections have been applied in this hardened revision.
