# Parameter Curation Council — v0.4.1

> **Date:** 2026-05-10
> **Council:** Research Software Engineer (AI-assisted)
> **Scope:** Deep literature audit of all MVP parameters

## Agenda

1. Audit physical constants (CONST).
2. Audit semiconductor/interface parameters.
3. Audit effective charge closure parameters (C-B).
4. Audit defect reaction-diffusion parameters.
5. Audit numerical parameters.
6. Issue tier recommendations.

## Deliberation Record

### Physical Constants (q, k_B, ε₀)

**Verdict:** CONST tier confirmed. These are exact by the 2019 SI
redefinition. No further action needed.

### Semiconductor Parameters (ε_r, D_it)

**Verdict:** T2 candidates identified but NOT used in current code.
ε_r(Si) = 11.7 and D_it = 10¹⁰–10¹² eV⁻¹cm⁻² are well-established.
These should be recorded in the registry but not injected into code
without implementing SI unit conversion first.

### Effective Charge Closure (λ_T, λ_C, C_sat, N_ref, t_eff)

**Verdict:** These remain TX or T3. The closure is Boussinesq-inspired
(ADR-007 Option C-B) and is inherently demonstrative. No physical anchor
exists for the coupling weights. N_ref could potentially be linked to
q·D_it but this requires unit infrastructure not yet available.

### Defect Parameters (D0_def, E_D, G0, R0)

**Verdict:** D0_def and E_D are promoted from T0 to T1 (order-of-magnitude
consistent with Si vacancy literature). G0 and R0 remain T0 (no anchor).

### Numerical Parameters (dx, dy, tolerance, max_iter)

**Verdict:** NUM tier confirmed. No physical calibration needed.

## Council Decision

> [!IMPORTANT]
> The v0.4.1 parameter curation is **approved as documentation-only**.
> No code changes are authorized. No parameter is promoted to calibrated.
> Future calibration requires ADR approval and unit conversion infrastructure.

## Dissenting Views

None recorded. The conservative approach (document but do not calibrate)
is unanimously endorsed.
