## Authorized Physics

```
N_it = ∫_{E₁}^{E₂} D_it(E) dE
```

Piecewise approximation:

```
N_it = Σᵢ D_i × ΔE_i
```

where `ΔE_i = E_{i+1} - E_i` and `D_i` is the trap density in bin `i`.

**Units:** [J⁻¹·m⁻²] × [J] = [m⁻²]

Downstream chain (unchanged):

```
σ_eff = s_charge × q_e × N_it × f_occ
```
