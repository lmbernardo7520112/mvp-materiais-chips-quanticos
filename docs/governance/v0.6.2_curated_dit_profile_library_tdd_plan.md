# v0.6.2 Curated D_it(E) Profile Library — TDD Plan

## Goal

Create curated E1/E2 D_it(E) profile library on top of v0.6.0
energy_profiles.py, using literature values already documented
in the repository.

## Authorized Scope

- Operational profiles must be E1 or E2.
- E1 profiles are literature-informed (scale ranges from existing docs).
- E2 profiles are experimental-profile-prior with complete metadata.
- S0_TEST_ONLY is allowed only inside tests.
- E0 is rejected.
- E3/E4 are blocked.

## Non-Goals

- no ρ_eff
- no t_eff
- no solver coupling
- no Poisson runtime coupling
- no Schrödinger
- no C2/C3
- no calibration claims
- no coherence prediction

## Future Module

`src/mvp_quantum_materials/dit_profile_library.py`
