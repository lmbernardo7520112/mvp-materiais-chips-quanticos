# v0.3 Implementation Plan — Defect-like Reaction-Diffusion

> **Status:** DRAFT — awaiting user approval; no implementation yet
> **Branch:** `research/v0.3-defect-like-council`
> **ADR:** ADR-006 (Proposed)

---

## Vision

Introduce a defect-like state variable C_def governed by reaction-diffusion
dynamics coupled to the existing thermal field, advancing the project
from numerical infrastructure toward semiconductor material modeling.

## Context

- v0.2.1 closed: thermal 2D robustness + institutional package
- ADR-005 roadmap: v0.3 = defect-like reaction-diffusion
- ADR-006 (Proposed): defines C_def, G(T), R(T,C)
- Maurand et al. (2016) is device-target anchor (not equation source)
- Equation structure inspired by: Dornberger (2001), Brown (1994),
  Voronkov (1999) — these are equation sources, not parameter sources
- All v0.3 parameters are toy/demonstrative unless curated from
  peer-reviewed literature with explicit justification

## Scope IN (MUST)

1. **defect_solver_2d.py** — 2D reaction-diffusion solver for C_def
   - Explicit Euler + CFL stability guard
   - One-way coupling: T(x,y,t) → C_def(x,y,t)
   - Neumann no-flux BCs
   - Logistic saturation term
2. **defect_config.py** or extension of config.py
   - Stability condition for reaction-diffusion + diffusion
   - Parameter defaults (toy/demonstrative)
3. **run_defect_2d.py** — CLI script
4. **Plots** — C_def contour plot, freeze-in profile
5. **CSV** — defect metrics export
6. **Tests** (TDD) — stability, boundedness, finitude, regression
7. **Convergence** — manufactured solution for reaction-diffusion
8. **Documentation** — parameters, report, walkthrough update

## Scope IN (SHOULD)

9. Sensitivity of C_def to key parameters (A_G, E_D, T_G)
10. Comparison figure: T(x,y) vs C_def(x,y) side-by-side
11. Integration into generate_all_results.py

## Scope OUT (WON'T)

- Poisson solver
- Schrödinger solver
- TCAD/QTCAD integration
- Two-way coupling (C_def → T)
- Full V/I coupled system
- Calibrated parameters
- Charge noise prediction
- Qubit coherence prediction
- Phase-field
- Czochralski real geometry
- OpenFOAM/FEniCSx/MOOSE

## Planned Tests

| Test | Purpose |
|------|---------|
| C_def remains non-negative | Boundedness |
| C_def ≤ C_sat | Upper bound |
| C_def finite | No NaN/inf |
| Solver rejects unstable dt | Stability guard |
| Zero G → C_def stays at IC | No spurious generation |
| High R → C_def decays | Recombination works |
| Output shape matches domain | Dimensional consistency |
| Neumann BCs enforced | Flux conservation |
| v0.1 tests pass | Regression |
| v0.2 tests pass | Regression |
| Coverage ≥ 70% | Quality gate |
