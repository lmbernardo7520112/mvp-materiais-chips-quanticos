# Acceptance Gates — v0.4.1 Parameter Curation

> **Date:** 2026-05-10
> **Status:** ACTIVE

## Gate Checklist

### G1: Registry Integrity

- [ ] `v0.4.1_parameter_registry.json` is valid JSON.
- [ ] Every parameter has a `current_tier` field.
- [ ] Every parameter has a `proposed_tier` field.
- [ ] Every parameter has `dimension` and `SI_unit` fields.
- [ ] Every T2/T3 parameter has at least one `source_id`.
- [ ] Every TX parameter has a `do_not_claim` field.

### G2: Literature Coverage

- [ ] At least 10 primary/review sources cited.
- [ ] Sources classified by type (PRIMARY_EXPERIMENTAL, REVIEW, TEXTBOOK, etc.).
- [ ] BibTeX file (`v0.4.1_sources.bib`) exists and is parseable.
- [ ] No sources from SiC, GaN, or HfO₂ used as Si/SiO₂ proxies.

### G3: Unit Audit

- [ ] C/m² vs C/m³ distinction documented for surface vs volume quantities.
- [ ] eV⁻¹ cm⁻² → J⁻¹ m⁻² conversion factor documented.
- [ ] cm²/s → m²/s conversion factor documented.
- [ ] t_eff role in surface-to-volume conversion documented.

### G4: Overclaim Audit

- [ ] No parameter described as "calibrated" or "validated".
- [ ] No claim of device-level prediction.
- [ ] No claim of charge noise prediction.
- [ ] No claim of qubit performance prediction.
- [ ] C_def explicitly described as dimensionless proxy.
- [ ] delta_rho_eff explicitly described as uncalibrated.

### G5: Scope Containment

- [ ] Zero changes to `src/`.
- [ ] Zero changes to `scripts/`.
- [ ] Zero changes to `tests/`.
- [ ] `policy.json` unchanged.
- [ ] Quality gates remain green.
- [ ] No new dependencies introduced.

### G6: Documentation Quality

- [ ] Decision brief exists with clear recommendation.
- [ ] Council deliberation is recorded.
- [ ] Risk matrix identifies key risks.
- [ ] Release notes draft exists.
- [ ] Walkthrough updated with v0.4.1 evidence.
