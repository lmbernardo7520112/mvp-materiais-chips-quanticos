## Scope

Implement a minimal Poisson bridge connecting process-level fields
(T, C_def) to electrostatic potential disorder (δφ) via a
Boussinesq-inspired trap-charge closure (Option C-B).

## Non-Goals

- No Schrödinger equation.
- No TCAD/QTCAD.
- No coherence, fidelity, or noise predictions.
- No calibration to experimental data.
- No real device geometry.

## Tests-First Plan (TDD)

### effective_charge.py Tests

1. `test_zero_cdef_gives_zero_perturbation` — h(0) produces no charge perturbation.
2. `test_uniform_cdef_gives_zero_delta_rho` — mean subtraction eliminates uniform field.
3. `test_delta_rho_integrates_to_approximately_zero` — charge neutrality at domain scale.
4. `test_monotonic_h` — h(C_def) is monotonically increasing.
5. `test_bounded_h` — h(C_def) ∈ [0, 1].
6. `test_lambda_sensitivity` — increasing λC increases δρ_eff magnitude.
7. `test_thermal_sensitivity` — increasing λT with non-uniform T changes δρ_eff.
8. `test_rejects_negative_dit_ref` — input validation.
9. `test_output_finite` — no NaN/Inf.
10. `test_output_shape` — correct array dimensions.

### poisson_solver_2d.py Tests

1. `test_manufactured_solution_quadratic` — known φ(x,y) = x²+y² → ρ = −4ε.
2. `test_manufactured_solution_sinusoidal` — φ = sin(πx)sin(πy).
3. `test_zero_source_constant_bc` — δρ=0, uniform BC → φ=const.
4. `test_neumann_compatibility` — ∫δρ dA = 0 for pure Neumann.
5. `test_dirichlet_boundary_values` — BC enforcement.
6. `test_output_shape` — correct dimensions.
7. `test_output_finite` — no NaN/Inf.
8. `test_convergence_with_refinement` — error decreases with finer grid.
9. `test_symmetry_preservation` — symmetric source → symmetric potential.
10. `test_rejects_incompatible_grid` — input validation.
