Create a small deterministic demo for C3 conservative grid projection.

* sign preservation;
* C2 metadata preservation;
* boundary metadata presence;
* calibration_status = not_calibrated;
* physical_interpretation_allowed = false;
* solver_coupling_enabled = false.

* no potential;
* no phi;
* no dummy phi;
* no physical field interpretation;
* no solver coupling;
* no quantum confinement;
* no AI-for-Science runtime;
* no calibration;
* no device prediction.

## Outputs

* c3_projection_demo.csv;
* c3_projection_demo_summary.json.

## Scope

This is a sanity/demo layer only. It does not validate device physics.
