## Scope: Option B

Option B introduces **dimensional scaffolding only**:

- Physical constants (ε₀, q) available as named constants.
- Relative permittivity lookup by material (Si, SiO₂).
- Absolute permittivity helper: ε = ε_r · ε₀.
- Scale mode infrastructure: demonstrative (default) vs literature-scaled.
- Geometry mode metadata: normalized_2d (default).
- Explicit labeling: φ is demonstrative unless all sources and geometry are physical.

Option B is **dimensional scaffolding**. It does NOT make φ physically
interpretable. If δρ_eff and domain geometry remain demonstrative, φ must
still be labeled demonstrative.

## NOT in Scope: Option C

The following are **explicitly excluded** from v0.4.4:

- D_it → D_it_SI conversion in code.
- delta_E_window calculation.
- σ_eff physical computation.
- ρ_eff physical/calibrated computation.
- t_eff physical justification or use.
- Physical domain geometry (Lx, Ly in meters).
- Charge sign convention implementation.
- Perturbation/neutralization mode implementation.
- Calibrated φ in volts.
- Device-level prediction.
- Schrödinger equation.
- Nonlinear Poisson.
- Self-consistent trap occupancy.

Physical constants and material property lookups.

```python
# Constants
EPSILON_0: float        # 8.854_187_8128e-12 F/m (CONST_DERIVED/CODATA)
ELEMENTARY_CHARGE: float  # 1.602_176_634e-19 C (CONST_EXACT)

# Functions
def relative_permittivity(material: str) -> float:
    """Return ε_r for a known material. Raises ValueError for unknown."""

def absolute_permittivity(epsilon_r: float) -> float:
    """Return ε = ε_r · ε₀. Raises ValueError if epsilon_r <= 0."""
```

**Supported materials:** `"Si"` (11.7), `"SiO2"` (3.9).

**Docstrings must state:** These are literature values, not device-specific
calibration. Using these constants does not constitute calibration.

### `src/mvp_quantum_materials/scale_modes.py`

Scale mode, geometry mode, and potential interpretation metadata.

```python
class ScaleMode(Enum):
    DEMONSTRATIVE = "demonstrative"
    LITERATURE_SCALED_CONSTANTS = "literature_scaled_constants"

class GeometryMode(Enum):
    NORMALIZED_2D = "normalized_2d"

class PotentialInterpretation(Enum):
    DEMONSTRATIVE = "demonstrative"
    DIMENSIONAL_SCAFFOLDING = "dimensional_scaffolding"

@dataclass
class ScaleMetadata:
    scale_mode: ScaleMode
    geometry_mode: GeometryMode
    source_mode: str            # "demonstrative" or "literature_scaled"
    phi_interpretation: PotentialInterpretation
    phi_unit_label: str         # "demonstrative (a.u.)" or "V (scaffolding, not calibrated)"

    def physical_interpretation_allowed(self) -> bool:
        """True only if ALL of scale, geometry, and source are physical."""
```

**Default:** `ScaleMode.DEMONSTRATIVE`, `GeometryMode.NORMALIZED_2D`,
`source_mode="demonstrative"`, `phi_interpretation=DEMONSTRATIVE`,
`phi_unit_label="demonstrative (a.u.)"`.
