# v0.7.6 AI-for-Science Future RED Plan

## Trigger

Only if a future ADR accepts an isolated AI-for-Science prototype. This plan is not activated in v0.7.6.

## Future First Prototype

Not C1/C2/C3.

The first prototype must be an analytic benchmark only — a problem with a known exact solution that can be used to verify the correctness of the neural approximation independent of project physics.

**Examples:**

- 1D Poisson equation with known analytic solution (e.g., u'' = -sin(πx), u(0)=u(1)=0).
- Simple 1D diffusion equation with known solution (e.g., heat equation with Gaussian initial condition).
- Manufactured solution benchmark (chosen solution, compute source term, verify recovery).
