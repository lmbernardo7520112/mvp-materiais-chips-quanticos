# v0.5.0 C1 Policy Activation

> **Date:** 2026-05-14
> **Authorization:** ADR-009 Accepted (v0.4.10)

---

## Reason

The C1 implementation adds `surface_charge.py` and `test_surface_charge.py`.
These files must be recognized by the scope guardrails because the tests
mention controlled terms ("Poisson", "rho_eff") as **negative guards** —
assertions that prove these concepts are absent from C1.

## Authorization Source

ADR-009 Accepted in v0.4.10 (PR #29, tag v0.4.10).

## Authorized C1 Scope

- D_it conversion (eV⁻¹·cm⁻² → J⁻¹·m⁻²).
- δE_window explicitness.
- N_it areal density.
- σ_eff surface charge density.
- C1 bookkeeping only.

## Still Forbidden

- ρ_eff (volume charge density).
- t_eff (effective thickness).
- Solver coupling.
- ε substitution.
- Physical φ interpretation.
- C2/C3.

## Why Tests Mention Controlled Terms

`tests/test_surface_charge.py` mentions "Poisson" and "rho_eff" **only**
as negative guards to prove they are absent from the C1 module:

- `test_c1_does_not_create_rho_eff` — asserts `compute_rho_eff` is absent.
- `test_c1_does_not_modify_solver` — asserts "poisson_solver_2d" is not
  in the source code of `surface_charge.py`.

These are audit tests, not implementations.

## Stage Semantics (GREEN 2.1)

The v0.5.0 C1 implementation is governed by policy stage **v0.5**.

`current_stage` was changed from `"v0.4"` to `"v0.5"` to reflect that C1
surface-density bookkeeping is a new physical capability, not merely a v0.4
metadata extension.

The v0.5 stage:

- Inherits all v0.4 protections (forbidden terms, conditional terms, artifacts).
- Adds ADR-009 as a required ADR (Accepted).
- Adds `surface_charge.py` and `test_surface_charge.py` to authorized files.
- Adds the ADR-009 document to required docs.
- Preserves v0.4 as a historical stage (unchanged from its original form).

The v0.5 stage authorizes only:

- D_it conversion.
- δE_window explicitness.
- N_it computation.
- σ_eff computation.

The v0.5 stage does **not** authorize:

- ρ_eff.
- t_eff.
- Solver coupling.
- ε substitution.
- Physical φ interpretation.
- C2/C3.
